# SnippetBox 实现任务列表

## 项目概述

SnippetBox 是一款面向开发者的轻量级代码片段管理工具，采用本地优先架构，支持智能搜索和可选的云同步功能。

**开发周期**：4 周
**技术栈**：Electron + React + TypeScript + SQLite + FastAPI + PostgreSQL

## 任务组织

任务按照 4 周时间线组织：

- 第 1 周：基础架构与本地核心功能
- 第 2 周：智能搜索和导出功能
- 第 3 周：云同步和分享功能
- 第 4 周：测试、优化与交付
